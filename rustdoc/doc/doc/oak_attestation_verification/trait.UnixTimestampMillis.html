Expand description

Return a milliseconds-since-the-epoch timestamp value.

Endorsement validity structures in our JSON-based endorsements use milliseconds resolution, but [OffsetDateTime] provides only seconds or nanoseconds since the epoch.

This bridges a convenience gap, and helps with readability of code that works with validity times.
