pub struct LookupDataManager<L: OakLogger + Clone> { /* private fields */ }
Expand description

Utility for managing lookup data.

LookupDataManager can be used to create LookupData instances that share the underlying data. It can also update the underlying data. After updating the data, new LookupData instances will use the new data, but earlier instances will still used the earlier data.

LookupDataManager maintains the invariants consistent view on lookup data , and shared lookup data

Note that the data is never mutated in-place, but only ever replaced. So instead of the Rust idiom Arc<Spinlock<T>> we have Spinlock<Arc<T>>.

In the future we may replace both the mutex and the hash map with something like RCU.
