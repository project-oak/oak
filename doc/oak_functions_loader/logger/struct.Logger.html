pub struct Logger { /* private fields */ }
Expand description

A simple logger that splits logging between writing logs that contain only public, non-sensitive content and writing logs that could potentially contain sensitive content.

Writing of potentially sensitive content will be ignored unless the ‚Äúoak-unsafe‚Äù feature is enabled.
