Get root X.509 certificate used by the Proxy Attestation Service to sign certificates. This certificate is generated insode the TEE and is not the same certificate that is used to connect to the proxy itself.

Note: Current implementation doesn’t support sealing, so the proxy generates a new certificate on startup.
