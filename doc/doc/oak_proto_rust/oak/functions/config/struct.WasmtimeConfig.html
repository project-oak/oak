Expand description

Settings specific to the Wasmtime engine.

These fields are explicitly marked as optional so that we could detect their presence (as sometimes you do want to specify zero as a value). If a field is left unspecified, it will leave whatever Wasmtime has as the default.

See https://docs.rs/wasmtime/latest/wasmtime/struct.Config.html for more details.

§pooling_strategy: Option<PoolingAllocationConfig>

If specified, switches to a pooling allocation strategy. If not specified, leaves as default (on demand). https://docs.rs/wasmtime/latest/wasmtime/struct.Config.html#method.allocation_strategy
