Struct sev_serial::SerialPort

source · 
pub struct SerialPort { /* private fields */ }
Expand description

Basic implementation that allows for writing to a serial port using the SEV-ES and SEV-SNP GHCB IOIO protocol, or using direct port-based IO, depending on which IO port factory is used in the wrapper enum.

See section 4.1.2 in https://www.amd.com/system/files/TechDocs/56421-guest-hypervisor-communication-block-standardization.pdf for more information on the GHCB IOIO protocol.
