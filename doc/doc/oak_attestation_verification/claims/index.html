Contains structs for specifying in-toto statements and claims about software artifacts. See also https://github.com/project-oak/transparent-release/blob/main/pkg/claims/claim.go, https://github.com/project-oak/transparent-release/blob/main/pkg/intoto/intoto.go.

    Inner type for a simple claim with no further fields.
  • Statement
