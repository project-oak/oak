Functions§

  • extract_evidence
    Extracts measurements, public keys and other attestation-related values from the evidence without verifying it. For most usecases, this function should not be used. Instead use the verify function, which verifies the attestation and only returns evidence upon successful verification. Hence marked as dangerous.
