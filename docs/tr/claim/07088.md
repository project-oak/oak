# Type

Claim ID: 07088\
Link to self:
https://github.com/project-oak/oak/blob/main/docs/tr/claim/07088.md

Very generic claim which supports to distinguish between various types via an
URI parameter. The parameter must be present and takes the following form:

```text
https://github.com/project-oak/oak/blob/main/docs/tr/claim/07088.md?type=my_type
```

where `my_type` is a placeholder for an actual type. The purpose is to be able
to make a distinction between various types of TR endorsement packages signed
with the same developer key, since in such a scenario the endorscope list-by-key
or list-by-keyset features won't be able to tell them apart.

There are no steadfast rules about how to use this claim - it is left entirely
to the client to shape this. The client may also choose to create their own
custom type claims instead of using the present one.
