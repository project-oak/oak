# MPM Version

Claim ID: 31545\
Link to self:
https://github.com/project-oak/oak/blob/main/docs/tr/claim/31545.md

Claim which conveys the MPM version as an URI parameter. The parameter
must be present and takes the following form:

```text
https://github.com/project-oak/oak/blob/main/docs/tr/claim/31545.md?mpm_version_id=1-234
```

where `1-234` is a placeholder for an actual MPM version ID.
